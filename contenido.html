Componentes

Procesador-CPU

Función: Ejecuta las instrucciones de los programas y gestiona todas las operaciones lógicas y aritméticas.

Placa Base

Función: Interconecta todos los componentes entre si y permite que se comuniquen entre sí mediante circuitos, buses de datos y controladores.

Memoria RAM

Función: Almacena temporalmente los datos e instrucciones que la CPU necesita mientras está funcionando.

Almacenamiento-Disco Duro o SSD

Función: Guarda de forma permanente el sistema operativo, las aplicaciones, los datos de los usuarios y los registros del servidor.

Tarjeta de interfaz de red-NIC

Función: Permite la conexión del servidor a la red local (LAN) o a Internet.

Fuente de alimentación

Función: Convierte la energía eléctrica de corriente alterna (CA) en corriente continua (CC) para alimentar los componentes internos.
